## Run Locally

Clone or download the repository, then open the project folder.

```bash
python3 -m http.server 5173
```

Open in browser:

```text
http://localhost:5173
```

## Deployment

This is a static HTML, CSS, and JavaScript website. No build command is required.

### Netlify Settings

```text
Build command: leave empty
Publish directory: .
```

### GitHub Pages Settings

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```
